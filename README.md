This repository contains the DIY Electronics Workbench Pro, which is a virtual simulation playground for designing and testing analog electronic circuits directly inside your web browser.

With this application, you can drag and drop common components like transistors, resistors, capacitors, and active power sources onto an infinite grid canvas. You can draw, bend, and route wires cleanly around parts using Unreal Engine-style reroute nodes. The simulation uses real mathematical circuit equations to instantly show voltages, currents, and power draw across your layout. It also includes built-in tutorial guides to help you learn electronics step-by-step.

How to use it:
To run the workbench, simply open the index.html file in any modern web browser. You do not need to install any external software or packages. To run the automated quality checks before deploying your changes, open a terminal window in the project folder and run the check.sh command. This will verify that all required files exist and contain valid code.
